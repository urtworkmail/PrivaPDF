# PrivaPDF – 100% Free, Offline, Open-Source PDF Toolkit

> Your PDFs. Your Privacy. No uploads. No limits. No tracking.

PrivaPDF is a complete PDF toolkit that runs **entirely in your browser**. Unlike iLovePDF (217M visits/month) and Smallpdf, nothing ever leaves your computer.

## Why the world needs this

- **Real pain**: Existing tools upload your private documents to their servers, limit free uses, and lock features behind paywalls
- **Massive demand**: iLovePDF ranks #115 globally with 217M monthly visits; Smallpdf gets 36M+
- **Privacy crisis**: Students, freelancers, job seekers in Pakistan, India, and worldwide need to handle sensitive documents (CNIC, resumes, contracts) without risking data leaks
- **Better & free**: 100% client-side using pdf-lib and PDF.js – works offline after first load

## Features (v1.0 - working now)

1. **Merge PDFs** – Drag, reorder, combine
2. **Split PDF** – Extract pages by range
3. **Compress PDF** – Reduce size with quality control
4. **Images → PDF** – JPG/PNG to PDF
5. **PDF → Images** – Export pages as JPG/PNG
6. **Rotate PDF** – Fix orientation

All processing happens locally. No server. No API keys.

## Tech Stack

- Single HTML file (no build step)
- Tailwind CSS
- pdf-lib.js + PDF.js
- Works on GitHub Pages, Netlify, or just double-click index.html
